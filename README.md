# CG Beautifier / 过场 CG 美化

SillyTavern UI Extension：把 `<CG>...</CG>` 或 `[CG]...[/CG]` 包裹的内容渲染成沉浸式过场卡片，支持流式预览与用户设置背景图。

## 安装

把整个 `cg-beautifier` 文件夹放到：

```text
data/<你的用户 handle>/extensions/cg-beautifier
```

然后刷新 SillyTavern，在 Extensions 面板中启用 `CG Beautifier`。

## 推荐提示词

```text
你可以在回复中生成“过场 CG”，但必须遵守：
1. 仅当故事地点发生明确变化、场景进入新区域、或世界观静态设定需要短暂展示时使用。
2. 每轮回复至多生成一次。
3. CG 必须使用以下格式，标签独占一行：
<CG>
【简短标题】
1-3 句简短、静态、具有镜头感的设定描述。
</CG>
4. CG 内容只允许展示静态设定：地点风貌、世界观规则、异能机制、装备设定、社会组织、历史遗迹、环境氛围。
5. 禁止在 CG 中展示人物性格、人物关系、人物动机、心理活动、好感度、秘密立场、剧情结论或对话。
6. 不要为了装饰而频繁生成 CG；没有地点/场景/静态设定变化时，不生成 CG。
```
